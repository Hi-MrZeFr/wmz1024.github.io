如何使用github page做个网页

我的博客|

前提

你得会html<最基本了>
css和js是次要

问题

那你可能就问了
你：为什么不用hexo呢
那是博客 下次讲

你：为什么md不行
我又没说不行

准备

啊这 不会吧 不会吧 不会有人还以为不用准备吧 <攒字数>

一个github账号一个
电脑一个<手机也可>
一个网络<不要用屏蔽github的网络>

开始

第一步

首先进入这里新建一个库
库的名字是 your-github-name.github.io |<不要原样照抄，不然很惨>
不会吧 不会吧 这都不会 把your-github-name改成你自己的github名字

下面有个readme.md 记得点上 <不点也是没关系的>
点击新建库

第二步

进入库的管理页面之后
点击 add file 之后点击create file 新建一个叫 index.html文件
如果你会html 你可以随便弄点 如果你不会 你就随便输入点文字 也可以按照我的代码

老hello world了
之后打开setting 找到github page 有一个网址 <通常是yourgithubname.github.io格式>

使用

点开网址 因为github服务器在外国 所以会有点慢

其他

因为本人m↓刚学 所以还不熟 如果你有更好的方法可以和我的 邮箱联系
