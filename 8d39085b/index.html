如何安装git

uh 这都不会安装 但是u1s1 在国内除非你找taobao镜像 不然下载都是个难题

安装

windows

首先下载git
众所周知git下载服务器在国外
众所周知还有npm镜像
哈
下载:here
降序排列 最底下的是最高版本
这里以2.28.0版本为例
下载好按照他的步骤就好了
之后就可以使用了

liunx

liunx可真是个大类呢

termux | linux

pkg install git

yum | linux

yum -y install git

ubuntu&debian | linux

sudo apt-get install git
不用sudo可能安装不了
如果你已经是root su账户了 就直接输入

老版本ubuntu&debian | linux

sudo apt-get install git-core

因为以前有个软件也叫GIT（GNU Interactive Tools），结果Git就只能叫git-core了。由于Git名气实在太大，后来就把GNU Interactive Tools改成gnuit，git-core正式改为git。

这句转自here

mac os

去Appstore下载Xcode
Xcode里面会集成git
