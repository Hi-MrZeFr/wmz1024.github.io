如何使用Windows电脑开个minecraft JE服务器

准备

一个电脑 RAM2GB以上
一个内网穿透账号 有公网这个当我没说

开始

第一步<安装过java的人请忽略>

安装个java
下载就行了
安装之后
右击此电脑
点击属性
点击高级系统设置
点击环境变量
在系统变量的一栏里
新建一个变量
叫JAVA_HOME
添加你的java安装路径
之后找到path变量<忽略大小写，如果没有此变量请新建>
改成 %JAVA_HOME%\bin; %JAVA_HOME%\jre\bin
找到classpath 没有就新建 不注意大小写
添加 .;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar
之后 打开cmd
输入 java
之后不显示没有此命令代表可以了

第二步

首先前往这里下载一个server.jar
在一个随便的目录新建一个文件夹<最好是英文文件夹 不然会很惨 >
之后打开
把server.jar移动到此文件夹
之后打开powershell
cd 到你的目录
输入

Xmx 表示的是最小内存
Xms 表示的是最大内存
最大内存推荐2048M以上
最小内存推荐1024M左右
输入下去之后
如果报错了请尝试下面的方法
如果没报错直接往下翻到第三步

2.1如果你失败了

直接点击server.jar
等一会会出现些文件
把eula.txt打开
把FALSE修改成true
之后再次点击server.jar
会出现相当多的文件
不要再点下去了
运行cmd
cd 到服务器目录
输入

之后输入help获取可以弄得指令
看2.1的朋友请调到4步

第三步

之后powershell会闪退下是正常现象
在 服务器目录 找到eula.txt
把FALSE改成true
关闭并保存
找到cmd
cd 到服务器目录
输入

之后就可以输命令了
但是这还没完
还得内网穿透

第四步

进入natfrp樱花映射
登录账号
点击用户信息
找到访问密钥选项
显示
之后复制
之后去软件下载
下载一个版本
不要急着打开
找到新建隧道
找一个服务器
本地端口写25565
其他都不用填
确定即可
之后把下载下来的压缩包
解压
我这里弄得是图形版
把刚才复制的密钥填进去
确定
找到隧道选项卡
勾选上刚才新建的隧道
记录好ip+端口
直接链接就好了(●°u°●)​ 」
